FIFA World Cup 2026 Penalty Simulator

GitHub Live Demo
Python · XGBoost · scikit-learn · FastAPI · Streamlit · Docker · Hugging Face Spaces

A machine-learning pipeline that predicts penalty-kick outcomes for FIFA World Cup 2026 takers and keepers — trained on over a thousand historical penalties from StatsBomb open data, with features engineered around each taker's tendencies and each keeper's strengths.

Combines a shot-placement model with an outcome predictor, calibrated so the predicted goal probability stays within a couple of percent of reality. Wrapped in a visual matchup dashboard with player photos, served by a FastAPI backend and a Streamlit frontend deployed across Hugging Face and Streamlit Cloud.

Resume ↔ Job Description Matcher

GitHub Live Demo
Python · sentence-transformers · Aho-Corasick · Gemini API · ESCO taxonomy · Streamlit · PyMuPDF

A tool that reads a resume and a job description, figures out which requirements you already cover (and how strongly), and tells you exactly what's missing — backed by the ESCO skills taxonomy and semantic embeddings rather than naive keyword matching.

It then uses an LLM to rewrite weak resume bullets, but only when the experience is genuinely implied by an existing bullet — so the model can't fabricate qualifications you don't have. Built in Python with sentence-transformers, Gemini, and Streamlit.
